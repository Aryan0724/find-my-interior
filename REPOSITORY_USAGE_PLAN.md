Version: 1.0

Purpose:
Use existing open-source repositories and UI libraries to reduce development time from months to weeks.

---

# MASTER STRATEGY

DO NOT BUILD EVERYTHING FROM SCRATCH.

FindMyInterior is a combination of:

* Houzz
* Urban Company
* IndiaMART
* Buildertrend
* Architizer
* Urban Ladder

Building these systems from scratch is impossible within project constraints.

We will use existing repositories and customize them.
